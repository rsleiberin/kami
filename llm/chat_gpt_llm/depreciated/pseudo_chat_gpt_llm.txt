ChatGPT Integration with Project Kami
-------------------------------------

1. Import required libraries and modules
   - OpenAI API
   - Project Kami's SubAgentRegistry and ContextManager

2. Initialize ChatGPT Assistant Class
   - API Key setup
   - Initial system message setup

3. Methods
   A. `send_to_chat_gpt(context, user_message)`
      - Input: Current context from Project Kami, User Message
      - Action: Make API request to ChatGPT
      - Output: ChatGPT's Response
      
   B. `receive_from_chat_gpt(chat_gpt_response)`
      - Input: Response from ChatGPT
      - Action: Parse the response and update Project Kami's context or trigger functions
      - Output: None

   C. `run()`
      - Loop for continuous conversation
      - Calls `send_to_chat_gpt` and `receive_from_chat_gpt` in sequence

4. Main Execution
   - Create an instance of ChatGPT Assistant Class
   - Call the `run()` method for continuous interaction
